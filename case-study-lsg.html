Living Style Guide

Process Gallery

The design was based on atomic UX design. This breaks down each component into "atoms". The atoms can then be used to build "molecules". For example, a button, font, and input field can be used to create a submission form. The guide is generic enough to be used by any brand, and is effortlessly scalable.

The brand, "Seamster", (a personal side project idea) was used as an example. Shown is a style tile specific to the brand

Sass was used with partials to organize the file structure.

Each atom is represented with it's own view/route in Node
